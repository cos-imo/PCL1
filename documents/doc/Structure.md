# Structure

## Arborescence primaire

```
.
├── Documents				* Dossier comprenant tout les documents relatifs à notre projet: documentation, cahier des charges...
│   ├── Cahier des charges		* Dossier contenant le cahier des charges
│   ├── Supports pédagogiques		* Dossier contenant les documents mis à disposition par l'équipe enseignante
│   └── doc				* Ensemble de la documentation écrite par les membres du projet
├── brouillon_ZHOU
└── source				* Dossier contenant tout le code
    ├── argument			* Module de chargement du fichier depuis la ligne de commande
    ├── automate			* Module de création d'automate
    ├── aux				* Utilitaires
    ├── token_analyser			* Module d'analyse lexicale - reconnaissance des tokens
    ├── token_generator			* Module d'analyse lexicale - découpage du texte en token
    └── variables			* Module recensant les variables du projet
```
